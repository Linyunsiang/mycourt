3.利用目的 / 利用目的（APPI 第17條）

  • 提供帳號、賽事報名、賽程編排、計分、家族（社團）管理等功能
  • 賽事主辦方確認參賽者身分、聯絡、保險與獎項寄送
  • 發送比賽提醒、家族通知等推播
  • 服務維運、客服、不當行為調查與防止
  • 統計分析以改善服務（去識別化後）

4.同意之取得 / 同意の取得

  • 首次啟動時，於登入前顯示本政策與《使用者授權合約（EULA）》並取得同意。
  • 報名大賽輸入高敏感個資前，再次顯示「個資蒐集同意」說明並取得明示同意。
  • 使用者可隨時撤回同意（見第 8 條）。

5.第三方提供與委外 / 第三者提供・委託

本服務不販售個資。為提供功能，將個資儲存／處理於下列服務商：

  • Google Firebase / Google Cloud（Authentication, Cloud Firestore, Cloud Messaging） — 資料儲存、登入、推播。
  • Google Sign-In / Sign in with Apple — 第三方登入驗證。
  • PayPay — 大賽報名費之線上收款（導向 PayPay 付款頁，本服務不儲存信用卡號）。

6.國外傳輸 / 外国にある第三者への提供

上述 Firebase / Google Cloud 之伺服器可能位於日本以外（如美國）。 使用者同意本政策即表示同意將個資傳輸並儲存於這些境外伺服器（依 APPI 第28條）。

7.安全管理措施 / 安全管理措置

  • 傳輸全程 TLS 加密；資料庫採嚴格存取權限（Firestore Security Rules，預設拒絕、僅本人／權限者可存取）。
  • 僅必要人員可存取後台；定期檢視權限。

8.當事人權利與帳號刪除 / 本人の権利・アカウント削除

使用者可行使開示、訂正、利用停止、刪除等權利：

  • App 內帳號刪除：マイ → 設定 → アカウント削除，將刪除帳號及關聯個資（戰績、家族成員身分、報名資料等）。
  • 其他請求請聯絡第 1 條之窗口，我們將於合理期間內回覆。
  • 部分資料可能因法令（如交易紀錄）需保存一定期間後刪除。

9.保存期間 / 保存期間

個資於達成利用目的或使用者刪除帳號後，於合理期間內刪除或去識別化（法令另有規定者除外）。

10.兒童之個資 / 子どもの個人情報

本服務面向一般使用者。未滿 16 歲者使用時，應取得法定代理人（親權者）同意（依 APPI 2025 修法方向）。

11.政策變更 / 改定

本政策修訂時將於 App 內或本頁公告。重大變更將另行通知並視情況重新取得同意。
